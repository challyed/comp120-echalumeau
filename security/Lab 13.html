Introduction

This report is a vulnerability analysis of the website Not Uber located on https://jordan-marsh.herokuapp.com/. This report aims to locate, analyze, and resolve any vulnerability abilities the site might have.

Methodology

For this lab, OWASP ZAP version 2.10.0 and the Burp Suite Community Edition v2021.2.1 were used to record and log HTTP(S) traffic and intercept requests and responses HTTP requests header fields and request body including query strings and data. Alternative applications that can be used are Tamper Data for Firefox and mitmproxy. One thing to note, even if the applications say that there is a vulnerability it does not mean that it is there. It could be a false positive.

Abstract of Findings

During the analysis, three issues were discovered. There was DOM-related cross-site scripting issue, a Server Leaks Information via "X-Powered-By" HTTP Response Header Field(s) issue and then there was an x.
